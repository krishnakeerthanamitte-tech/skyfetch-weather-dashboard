# skyfetch-weather-dashboard
# 🌤️ SkyFetch - Weather Dashboard

A beautiful and interactive **weather dashboard** that provides **real-time weather data and a 5-day forecast** for any city in the world.

This project demonstrates modern **JavaScript concepts, API integration, and responsive UI design** while building a practical web application.

---

# ✨ Features

🔍 Search weather for **any city worldwide**
🌡️ View **current temperature and weather conditions**
📊 **5-day weather forecast** with daily predictions
💾 **Recent searches saved locally** using localStorage
🔄 **Auto-loads the last searched city** on refresh
📱 **Fully responsive design** (desktop + mobile)
⚡ Fast API calls using **Axios**

---

# 🛠️ Technologies Used

* **HTML5**
* **CSS3** (Flexbox, Grid, Animations)
* **JavaScript (ES6+)**
* **Axios** for API requests
* **OpenWeatherMap API**
* **localStorage** for saving recent searches

# 💻 Local Setup

Follow these steps to run the project locally.

### 1️⃣ Clone the repository

```bash
git clone https://github.com/YOUR-USERNAME/skyfetch-weather-dashboard.git
```

### 2️⃣ Navigate into the project folder

```bash
cd skyfetch-weather-dashboard
```

### 3️⃣ Get a free API key

Go to:

https://openweathermap.org/api

Create an account and generate a free API key.

### 4️⃣ Add your API key

Open **app.js** and replace:

```javascript
const app = new WeatherApp("YOUR_API_KEY");
```

with your actual API key.

### 5️⃣ Run the project

Simply open:

```
index.html
```

in your browser.
